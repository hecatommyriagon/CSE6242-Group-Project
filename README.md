# CSE6242-Group-Project

0. Make sure you are in the project root (ie `./CSE6242-Group-Project`)
1. Use python version 3.10.0 pls (or at least 3.10.x)
2. Recommended: Make a new python virtual environment via `python3.10 -m venv .venv` and activate it via `source .venv/bin/activate`
3. Install python packages via `pip install -r requirements.txt`
4. To run web app, run `python app.py` then navigate to `http://127.0.0.1:5000` on your web browsrer
